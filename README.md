### 프로젝트 폴더로 이동
```bash
cd flutter_console_shopping_mall
```

### 실행
```bash
dart run
```

### 구현된 필수 기능
1. 판매하는 상품 목록을 볼 수 있음
2. 상품을 장바구니에 담을 수 있음
3. 장바구니에 담은 상품들의 총 가격을 볼 수 있음
4. 쇼핑몰 프로그램을 종료 할 수 있음

### 구현된 도전 기능
1. 쇼핑몰 프로그램을 종료할 시 한번 더 종료할 것인지 물어봄
2. 장바구니를 초기화할 수 있음
3. 장바구니에 담은 상품들의 목록과 가격을 볼 수 있음
